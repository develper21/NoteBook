# NodeJS REPL

## What is REPL?
- The NodeJS stands for Read-Eval-Print-Loop.
- It is a simple command-line interface for NodeJS.
- It's allows you to execute NodeJS code snippets directly in the terminal.
- It's very useful for quick testing and debugging of NodeJS code.

## How to start REPL?
- Open the terminal.
- Type `node` and press Enter.
- You will see the REPL prompt `>`.
- You can type NodeJS code and press Enter to execute it.
- To exit the REPL, type `.exit` and press Enter.

## REPL Commands
- `.help` - Display help message.
- `.break` - Exit the current input.
- `.clear` - Clear the current input.
- `.editor` - Enter editor mode.
- `.load` - Load a file into the REPL.
- `.save` - Save the current session to a file.
