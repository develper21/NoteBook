># NodeJS Introduction

## What is NodeJS?
- NodeJS is a free open-source, cross-platform JavaScript runtime environment.
- It is built on the V8 engine, which is the same engine that powers Google Chrome.
- and Thay allows developers to run JavaScript code outside of the web browser.

## NodeJS Features
- Server-side JavaScript
- Asynchronous and Non-Blocking I/O
- Built on the V8 engine
- Single threaded event loop
- Cross-platform
- Large ecosystem of packages
- Fast and lightweight
- Scalable

## NodeJS Use Cases
- Real-time Applications
- Web APIs
- Backend Services
- Microservices
- Streaming Applications
- Single Page Applications
- CLI Tools
